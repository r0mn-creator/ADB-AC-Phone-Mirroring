On the Android Device you need to enable developer mode. This video shows you how :
https://www.youtube.com/watch?v=zu9oCE9N8H0

Next, in develpoer options find ADB Debugging and turn it on.

Connect Device to PC and allow permission for the PC.

Next, Start the ADB Server using START.bat

If all the files are in place you should see the device in your Car when opening the mirror app.
